Form

O que é um Form:

A tag <form></form>(formulário) é utilizada para agrupar elementos interativos. Nela o usuário vai inserir insformações que serão enviadas ao servidor.

  1. Form (formulário)
  2. Label (etiqueta)
  3. Input (entrada)
  4. Button (botão)
  1. Primeiro crie a tag form que será responsável por agrupar todos os campos e buttons que iremos usar: <form> Conteúdo </form>
  2. Agora criamos uma tag label que é usada como legenda para a tag input que vamos usar: <label> for="name"> Nome </label> 
     
     <form>
       First Name: <input type="text" name="FirstName">  
        <br>
       Last Name: <input type="text" name="LastName">       
     </form>

Input

A tag input é como criamos o campo onde o usuário irá digitar as informações, esse campo pode ser de diferentes tipos: text, email, password. O input pode carregar vários atributos.

  • type: O tipo de atributo.
  • name: O nome dado ao input.
  • id: A identificação do input.
  • placeholder: o texto.

<input type="text" name="name" placeholder="Digite seu nome">



Input="text"

Cria um input onde pode ser adicionado texto

    
    <label for="name">Name</label>
    <input id="name" type="text" name="name" placeholder="Escreva o seu nome">

Input="email"

Cria um input usado exclusivamente para email

    
    <label for="email">E-mail</label>
    <input id="email" type="email" name="email" placeholder="Escreva o seu E-mail">

Input="number"

Cria um input exclusivamente para números

    
    <label for="numero">Idade:</label>
    <input type="number" id="numero" name="numero" placeholder="Escreva sua idade">

Input="password"

Cria um input onde pode ser adicionado senhas

    
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha" placeholder="Escreva o sua senha">

Input="radio"

Cria um input onde pode ser selecionado uma opção.

    
    <label for="aluno">Aluno vai na web</label>

    <input type="radio" id="sim" name="aluno">
    <label for="sim">Sim</label>

    <input type="radio" id="nao" name="aluno">
    <label for="nao">Não</label>

Input="checkbox"

Cria um input onde pode ser selecionado uma ou mais opções.

    
    <label for="batatas">Tipos de Batatas:</label>

    <input type="checkbox" id="frita" name="frita">
    <label for="frita">Batata frita</label>

    <input type="checkbox" id="assada" name="assada">
    <label for="assada">Batata Assada</label>

    <input type="checkbox" id="marechal" name="marechal">
    <label for="marechal">Batata de Marechal</label>

Input="submit"

    
    <input type="submit" value="Enviar">
